sinarngo.organization Installation
----------------------------------

To install sinarngo.organization using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.organization`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.organization

* Re-run buildout, e.g. with:

    $ ./bin/buildout
